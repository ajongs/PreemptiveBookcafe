# PreemptiveBookcafe
공주대학교 북카페 자리순환 서비스

### 공주대학교 북카페에 좌석신고 제도를 도입하여 좌석 사석화를 방지하고 북카페 사용률과 순환율을 증진시키기 위한 프로젝트입니다.

## 프로젝트 기술 스택
 - Java8
 - MySQL
 - JPA/Hibernate/Spring Data JPA
 - SpringBoot 2.6
 - Gradle
 - AWS EC2
 - Firebase FCM (Firebase Cloud Messaging)
